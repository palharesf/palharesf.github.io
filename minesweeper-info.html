SG Minesweeper

A Modern JavaScript Implementation of the Classic Game

About the Game

SG Minesweeper is my take on the timeless logic puzzle game that has been entertaining players since the early days of personal computing. This web-based version brings all the strategic gameplay you love, with a clean modern interface and smooth performance that works seamlessly across all devices.

The game challenges players to clear a rectangular board containing hidden mines without detonating any of them. Using clues about the number of neighboring mines in each field, players must logically deduce the locations of mines and flag them appropriately.

How to Play

Basic Rules

The objective is simple: reveal all cells on the board that don't contain mines. Each revealed cell shows a number indicating how many of the eight adjacent cells contain mines. If you reveal a cell containing a mine, the game ends.

Controls

  • Left Click: Reveal a cell
  • Right Click: Place or remove a flag to mark suspected mines
  • First Click Safety: The game ensures your first click never hits a mine, giving you a fair start every time

Strategy Tips

Success in Minesweeper comes from careful logical deduction. Start by identifying obvious patterns: if a cell shows "1" and touches only one unrevealed cell, that cell must contain a mine. Build outward from safe areas, using the numbers to create a mental map of mine locations.

Advanced players learn to recognize common patterns like "1-2-1" formations and corner configurations. When logical deduction runs out, probability becomes your friend - cells adjacent to fewer numbers are statistically less likely to contain mines.

Difficulty Levels

The game offers multiple difficulty levels to challenge players of all skill levels:

  • Beginner: 9x9 grid with 10 mines - perfect for learning the mechanics
  • Intermediate: 16x16 grid with 40 mines - for developing strategy
  • Expert: 30x16 grid with 99 mines - the ultimate challenge for experienced players
  • Custom: Create your own difficulty with custom board dimensions and mine counts

Technical Implementation

SG Minesweeper is built entirely with vanilla JavaScript, HTML5, and CSS3, with no external dependencies. This approach ensures fast loading times and reliable performance across all modern browsers. The game logic handles board generation, mine placement, recursive cell revealing, and win/loss detection entirely client-side.

Key technical features include:

  • Efficient flood-fill algorithm for revealing adjacent empty cells
  • Random mine distribution with first-click safety guarantee
  • Responsive design that adapts to different screen sizes
  • Clean, maintainable code structure for easy modification and learning
